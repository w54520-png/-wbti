## 0. 执行摘要（Executive Summary）

WBTI 是一款面向年轻人的**娱乐向酒人格测试**，参考 SBTI（Silly Big Personality Test）的"幽默 16+ 型人格测试"模板，做 32 道不直接谈酒的生活态度题，测出 16 个"酒人格代号"之一（4 字母英文 + 谐音梗 + 中文别称），结果页附带 3 张"代表酒"卡片作为引流钩子。

**MVP 范围**：HTML 简版（5 个静态页 + 1 个 JSON 数据），原生 HTML + Tailwind CDN，单页 hash 路由，部署到 Vercel/Netlify 静态托管。预期 1 周内上线，用于私域引流（公众号/微信群）和市场验证。

**关键决策**：
- 个人主体无法走 CPS 电商引流 → 用私域路径（公众号 + 微信群）替代
- 不做严肃心理测试 → 全程"娱乐向"口吻 + 底部"友情提示"免责
- 代号 4 字母以上 + 谐音/双关 + 酒元素
- 题目不直接谈酒 → 借鉴 MBTI 心理量表"伪装题"逻辑

## 1. 核心概念

### 1.1 一句话定位
> **「你的酒人格」** —— 32 道看似无关的有趣题，测出你是哪种酒 + 一份"专属酒单 + 同类酒友"。

### 1.2 调性
- 年轻人、自嘲、轻社交货币（让人想截图发朋友圈）
- 不卖弄专业（不是品酒师话术）
- 颜色：暖色系（深琥珀、浅金、米白、酒红）—— 避开"老气"的红金黑
- 参考 SBTI 的幽默克制口吻，但更"轻饮"（不夸张不低俗）

### 1.3 命名 WBTI 的解释
- **W**ine + **B**ig + **T**ype + **I**ndicator
- 也可解作 "What's your Bottle Type Indicator"（双关）
- 与 MBTI 4 字母呼应（让用户秒懂这是类 MBTI 测试）

## 2. WBTI 4 维 × 16 型代号设计

### 2.1 4 个维度（每维 2 极）

| 维度 | 一极 | 另一极 | 字母 |
|---|---|---|---|
| **浓度（Strength）** | 重（烈、浓郁、刺激） | 轻（顺、低度、清淡） | H / L |
| **场景（Scene）** | Solo（独酌） | Tribe（群嗨） | S / T |
| **风味（Flavor）** | Fruit（果香花香） | Malt（麦芽木质/泥煤） | F / M |
| **节奏（Rhythm）** | Classic（经典传统） | New（新潮现代） | C / N |

4 维度 × 2 极 = 16 个组合。每个组合对应一个"酒人格代号"。

### 2.3 代号命名逻辑（设计规则）
- **4 字母及以上**（参考 SBTI 的 4-5 字母规则：CTRL、THAN-K、DIOR-S、DRUNK）
- **酒元素关联**：每个代号本身是酒/饮酒相关英文词
  - 原料类：MALT（麦芽）、CASK（木桶）、HOPS（蛇麻草 - 备用）
  - 饮酒方式：NEAT（纯饮）、TIPS（tipsy 微醺）、BOOZE（俚语"酒"）
  - 风味：SMOK（烟熏）、PEAC（peach 桃，谐音 peace）
  - 容器：DECA（decanter 醒酒器 - 备用）
  - 鸡尾酒名：JULE（julep 鸡尾酒）、PUNC（punch 潘趣酒）
  - 单位：DRAM（少量烈酒单位）
- **谐音/双关**：英文原词 + 中文别称构成双层梗
  - MALT（麦芽）→ 麦霸（谐音"麦+霸"）
  - CASK（木桶）→ 老炮（cask 里陈年 = 老派）
  - NEAT（纯饮）→ 纯爷们（谐音）
  - JULE（julep）→ 茶花女
  - PUNC（punch）→ 胖次谐音（甜心感）
  - TIPS（tipsy）→ 蹦迪选手（tipsy 状态）
  - DRAM（少量烈酒单位）→ 深夜电台
  - SMOK（smoke）→ 烟嗓

### 2.4 隐藏人格机制（参考 SBTI）

| 隐藏代号 | 别称 | 一句话 | 触发条件 |
|---|---|---|---|
| **DRUNK** | 酒鬼 | 烈酒烧喉，不得不醉。 | 浓度 H 高分 + 饮酒分支选 A（最近 3 个月喝断片 ≥ 2 次） |
| **HHHH** | 傻乐者 | 哈哈哈哈哈哈。 | 4 维度所有常规人格最高匹配度都低于 60% |

## 3. 题目设计（32 道 + 1 道饮酒分支）

### 3.1 设计原则（核心）
借鉴 MBTI 心理量表的"伪装题"逻辑：
- ❌ **不出现** "酒/喝/味道/烈/淡/醉"等直接词
- ✅ **用生活场景/态度/价值观**做伪装
- ✅ 题目让人"觉得在测性格"——但答案悄悄映射到 4 维度
- ✅ 维度题打散：用户做完全程察觉不到 8+8+8+8 分组

### 3.2 量表
- 每题 **3 选 1**：A. 不认同 / B. 中立 / C. 认同
- 进度条 + 上一题 + 下一题（按固定顺序，无"重新随机题序"）
- 题与题之间 0.3s 切换动效
- 进度条用酒红色/琥珀色

### 3.3 32 道题（按最终展示顺序打乱）

> 注：实际页面展示顺序会在写代码时随机洗牌（但固定每次一致，避免结果混乱）

#### ——浓度维度（重 vs 轻）8 道——

1. 我愿意为了一道菜排 30 分钟队。
2. 跳伞、蹦极、过山车这类高刺激项目，我愿意尝试。
3. 比起清粥小菜，我更爱浓郁厚重的食物（红烧肉、奶酪火锅）。
4. 我愿意为了一场演唱会飞到另一个城市。
5. 我喜欢一口气把事情做到极致，不喜欢慢慢摸索。
6. 旅行时我最爱钻进当地小馆子，连锁餐厅让我扫兴。
7. 我喜欢让一首歌一直单曲循环，直到听吐。
8. 一份清淡的沙拉，很难让我感到满足。

#### ——场景维度（独 vs 群）8 道——

9. 周末夜晚，我更愿意一个人在家待着。
10. 我说话前会先想一下，很少插嘴。
11. 我有几个十年老友，比一堆饭局新朋友更让我踏实。
12. 比起热闹的酒吧，我更喜欢深夜一个人去居酒屋。
13. 出门吃饭我总爱坐包间或角落。
14. 深夜 12 点，我宁愿一个人安静做事。
15. 我很少在朋友圈发自拍。
16. 连续两天的聚会，会让我觉得被掏空。

#### ——风味维度（果香 vs 木质）8 道——

17. 比起纯黑咖啡，我更爱加了奶的拿铁。
18. 我喜欢花香味的东西（桂花茶、玫瑰饼、茉莉香水）。
19. 比起 90% 的黑巧，我更爱 60% 以下的牛奶巧克力。
20. 我偏爱甜口的菜（江浙菜、东南亚菜、甜辣酱）。
21. 我闻不习惯"老派香水的浓郁麝香味"。
22. 比起陈年普洱的苦，我更爱水果茶的清香。
23. 我觉得香水太浓不好闻，清淡的水生调更适合我。
24. 比起浓缩咖啡的苦，我更爱奶盖的绵柔。

#### ——节奏维度（经典 vs 新潮）8 道——

25. 比起新派歌手，我更喜欢老歌。
26. 我买东西更看重品牌历史，而不是新潮小众。
27. 装修我喜欢复古/中世纪/工业风，胜过极简/未来感。
28. 我看电影更爱老片/经典系列，而不是新上映的片子。
29. 我更爱陈年/老味道，而不是新工艺的清爽。
30. 比起当代艺术，我更爱传统油画/水墨画。
31. 我比较信"老字号"，觉得经得起时间的东西更可靠。
32. 比起小众独立游戏，我更爱经典 IP 系列。

### 3.4 饮酒分支触发题（第 33 题，条件出现）

仅当用户在浓度维度得 H（重口味）高分时插入：

> **"最近 3 个月，你有没有一次喝断片？"**
> - A. 有过，还不止一次
> - B. 有过一次
> - C. 从来没有

**A 选 → 触发 DRUNK 隐藏人格**。

### 3.5 计分规则

每题 A/B/C → -1/0/+1 给对应维度一极。

```
维度   一极   计分
浓度   H/L   题1-8, 33
场景   S/T   题9-16
风味   F/M   题17-24
节奏   C/N   题25-32
```

每维度极得分 = Σ 该维度相关题的分值（范围：-8 到 +8）

**最终人格判定**（按以下顺序判定）：

1. **饮酒分支触发判定**：仅当 H 维度得分 ≥ 2（8 题里 ≥ 2 题选 C/认同重口味倾向）时，第 33 题"饮酒分支"才出现
2. **隐藏人格 DRUNK**：第 33 题选 A（最近 3 个月喝断片 ≥ 2 次）→ 直接落到 DRUNK 酒鬼，跳过常规判定
3. **隐藏人格 HHHH**：4 维度常规人格的最高极得分 < 5（即最高极 < 8 题里的 5 题）→ 落到 HHHH 傻乐者
4. **常规人格判定**：4 维度各取得分最高的极 → 4 字母组合 → 查 16 型表（2.2 节）
   - 例：浓度 H 得分最高 + 场景 S 得分最高 + 风味 F 得分最高 + 节奏 C 得分最高 → HSFC → **MALT 麦霸**
5. **极得分平手处理**：当某维度两极得分相同时，默认选字母表靠前的极（H > L, S > T, F > M, C > N），并在结果页底部小字提示"该维度平手"

- 角色：抽象人形 + 酒元素道具（酒杯/酒瓶/橡木桶/麦穗）
- 每个 WBTI 代号一张角色图（共 16 张）

## 6. 技术栈

| 项 | 选择 | 理由 |
|---|---|---|
| 框架 | 原生 HTML + JS | 单页/少量页，无需打包 |
| 样式 | Tailwind CSS（CDN） | 改样式不用学 SCSS，直接改 class |
| 路由 | 单 HTML + JS hash 路由 | 静态部署友好，无需后端 |
| 数据 | JSON 配置文件 | 改文案只改 JSON |
| 部署 | Vercel / Netlify | 白嫖静态托管 + 自动 HTTPS + 绑域名 |
| 数据统计 | Vercel Analytics 或 Google Analytics | 看 PV/UV/完成率 |
| 域名 | 暂用 Vercel 子域名（wbti.vercel.app），未来买 wbti.cn | 域名注册约 ¥60/年 |

## 7. 未来扩展（不在 MVP 范围）

### 7.1 微信小程序化（v2.0）
- 个人主体可做"工具/测试"类小程序，但不能挂电商 CPS
- 引流路径：小程序测完 → 长按识别加公众号 → 公众号菜单点"加入酒友群"
- 需要：注册小程序账号（个人身份证）、备案（个人主体免 ICP）、提交审核
- 风险：测试类小程序审核严格，需要免责声明完整

### 7.2 引流落地
- **方案 A**（推荐）：在酒卡位置放公众号二维码（用户扫码关注）
- **方案 B**：注册个体工商户后接入京东/淘宝 CPS
- **方案 C**：和酒品牌方合作，用品牌方 CPS 链接

### 7.3 内容运营
- 公众号：每周发"酒人格 × 节日/季节"内容（如"圣诞夜的 WBTI Top 5"）
- 社群：每周发"本周酒单" + 用户晒图
- 裂变：测完分享生成"我的 WBTI 卡片" → 朋友扫码进入同测试

## 9. 风险与规避

| 风险 | 影响 | 规避 |
|---|---|---|
| 微信小程序审核拒绝 | 无法转小程序 | 免责声明完整；先做 HTML 试水 |
| 个人主体被认定违规 | 账号封禁 | 不挂电商链接；测试结果纯娱乐 |
| 题目"太准"被批评 | 心理测试争议 | 友情提示 + 文案娱乐化 |
| 16 张画风太贵/太久 | MVP 推迟 | 先 emoji 占位，v1.1 再升级 |
| 用户分享率低 | 传播不动 | 分享卡片设计是关键，单独迭代 |
| 数据合规问题 | 法律风险 | 不收集个人信息；不上传用户答案 |
