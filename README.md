# WBTI 酒人格测试

> 32 道看似无关的有趣题，测出你的酒人格。

## 项目状态

- v0.1 MVP (开发中)
- 部署地址：https://wbti.vercel.app（待上线）

## 本地运行

```bash
# 任意 HTTP server 即可（如 Python）
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```
